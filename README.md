# 🚀 GitHub AI Project Guide 

> **定位**：AI / 深度学习开发者专属的 GitHub 极简实战指南。

通用的 Git 教程很多，但针对 AI 开发者（涉及大模型权重、数据集、Colab 联动、多设备同步）的痛点指南却很少。本项目旨在为你提供一套**不冗余、易落地**的 Git/GitHub 全流程解决方案。

## 📂 仓库导航

- 📖 **[核心指南：GitHub & Git 基础到进阶](GitHub_AI_Guide.md)** - 涵盖所有高频实用的 Git 命令与协作规范。
- 🛠️ **[AI 专属配置模板 (templates/)](templates/)** - 开箱即用的 `.gitignore`（过滤模型权重与数据集）、项目 README 模板、环境配置脚本。
- 💡 **[高频报错速查 (faq/)](faq/)** - 专治各种网络报错、大文件推送失败、Colab 权限等疑难杂症。
- 💻 **[极简练手案例 (examples/)](examples/)** - 提供一个跑通全流程的最小化 AI 脚本。

## 🌟 核心亮点
- **Git LFS 大文件管理**：告别 `.pt`、`.safetensors` 等权重文件无法上传的烦恼。
- **Colab 无缝联动**：如何优雅且安全地在云端拉取、推送你的私有代码。
